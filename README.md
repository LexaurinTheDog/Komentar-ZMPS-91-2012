## Metoda

ZMPS se aplikuje **subsidiárně** k unijním nařízením (Brussels, Rome regimes, Brussels IV) a mezinárodním smlouvám. Komentář byl sestaven AI asistentem (Claude) **výhradně z lokálních materiálů**. **Necituje konkrétní spzn NS/NSS**, ale **cituje veřejně známé landmark SDEU věci** v oblasti kolizního práva a mezinárodní procesní (Centros, Überseering, Rome regimes interpretace).

Tento komentář **není autoritativním právním stanoviskem**. Pro praktickou aplikaci je nutné ověřit aktuální unijní nařízení a mezinárodní smlouvy (oblast se rychle vyvíjí — viz Brusel II ter 2019/1111 nahradil 2201/2003).
